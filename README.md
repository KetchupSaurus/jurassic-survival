# Jurassic Survival
By [KetchupSaurus](https://github.com/KetchupSaurus)

A dinosaur evolution survival game built with HTML5 Canvas and Electron.

## How to Play

Eat apples to evolve through 4 dinosaur stages. Survive against increasingly dangerous enemies and defeat bosses.

### Evolution Stages
| Stage | Dino | Apples Needed |
|-------|------|---------------|
| 1 | Gallimimus | 0 |
| 2 | Utah Raptor | 3 |
| 3 | T-Rex | 10 |
| 4 | Spinosaurus | 15 |

### Controls
| | Player 1 | Player 2 |
|--|----------|----------|
| Move | Arrow Keys | WASD |
| Attack | Space | Space / E |

### Enemies
- **T-Rex** — aggressive, chases you
- **Spinosaurus** — large and fast
- **Nano-T** — small but quick
- **Ankylosaurus** — slow and armored

### Bosses
- **Brontosaurus** — 1000 pts, 10 hits to defeat
- **Titanosaurus** — 2000 pts, 20 hits to defeat
- **Indominus Rex** — 3000 pts, 35 hits, shoots projectiles

## Running the Game

### In the browser
Just open `index.html` directly.

### As a desktop app
```bash
npm install
npm start
```

### Build Windows installer
```bash
npm run build
```
Output will be in the `dist/` folder.

## Two Player Mode
Select **2 Players** on the start screen. Both players share a score but have independent HP and evolution. The game ends when both players are defeated.
